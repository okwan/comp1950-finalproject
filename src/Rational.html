Layout Decisions


The most obvious improvement we have done is the overall UI enhancements. We provided a seamless color scheme, even adding text-highlighting color. The text on the current website becomes blurry and hard to read when you stare at it for a long time because of it's tight and small font. We wanted to make sure that the user does not get overwhelmed by information but focus on what they want to look at instead.


The biggest problems on the current website is that it can get overwhelming with all the info. It was also very troublesome to scroll through to access certain links. We strongly believe in the one-page vertical design still which was similiar to the current website. But we added better functionality by incorporating smooth scrolling to different exercises on the lecture page. Also, to access the homework, all you need to do is to click the floating nav button and click on the homework button to download it.


Listed above are all the deliverables that we implemented. The most advanced jQuery / CSS3 functions were the full screen overlays and smooth scrolling. With the powerful library of jQuery, it was quite easy to add / remove classes to do the overlay feature of opening and closing. We did take advantage of open-source libraries available such as animate, headroom, and prism.


Headroom was used to implement the scrolling nav feature, so the top navigation will be hidden when the user scrolls. This helps with overall usability and only show the navigation when the user really needs it. We implemented this because we believe the content is the most important thing that the user needs to be looking for.
